📀 bluray.com Import
bluray.com-ról exportált gyűjtemény importálása. EAN vonalkód vagy cím alapján egyezteti a meglévő filmekkel, hiányzó mezőket tölti ki (stúdió, Steelbook, lemezszám, kiadási dátum stb.).
📥 Importálás XLSX-ből
Importáld a meglévő Excel adatbázisodat. Az app megőrzi az összes adatot és kiegészíti képekkel, TMDB adatokkal.
☁️ Google Drive szinkron
Az adatbázis a Google Drive-odra mentődik (bluvault_db.json), így iPhone, Android és PC-n is ugyanaz az adat látszik.

Gyors javaslatok:
🎬 TMDB API beállítás
A The Movie Database API-val automatikusan tölt be posztereket, magyar címeket és film infókat. Ingyenes regisztráció: themoviedb.org
🖥️ Jellyfin integráció
Csatlakoztasd a Jellyfin szerveredet. A NAS-on tárolt filmek automatikusan szinkronizálódnak a gyűjteményeddel.
💡 Tailscale telepítése UGreen NAS-ra: Docker → Tailscale image → hálózati mód: host
Az API kulcsot a Jellyfin admin felületen találod: Dashboard → API Keys
📤 Exportálás
Exportáld a teljes gyűjteményt CSV vagy JSON formátumban
🔒 PIN zár
Védd az appot PIN kóddal. Minden megnyitáskor, és háttérbe kerülés után kéri.
🔲 Megjelenítés beállítása
Rácsnézet mérete és lista nézet. Az automatikus méret a képernyő szélességéhez igazodik.
